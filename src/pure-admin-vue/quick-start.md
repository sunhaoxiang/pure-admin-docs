# 快速开始

## 准备工作

::: tip 确保 Node.js 版本满足以下要求：
如果使用 `Node.js 20`，最低支持版本为 `20.9.0`

如果使用 `Node.js 22`，最低支持版本为 `22.11.0`
:::

在 Visual Studio Code 里安装以下扩展：

- ESLint
- DotENV
- Vue - Official
- UnoCSS
- Iconify IntelliSense
- i18n Ally

### 下载源码

```bash
git clone https://github.com/sunhaoxiang/pure-admin-vue.git
```

### 安装依赖


```bash
pnpm i
```

### 运行项目

```bash
pnpm dev
```
